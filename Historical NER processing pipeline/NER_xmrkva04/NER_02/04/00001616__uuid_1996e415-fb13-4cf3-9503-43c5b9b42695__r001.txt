Pro teplotu 148° C se užije cholesterinu.
Pro teplotu 159° C se užije acetnaftalidu.
Pro teplotu 165° C se užije acetylmočoviny.
Pro teplotu 170° C se nejlépe hodí 1,9 peribenzantron.
Do formy má vždy býti vpravena přesná dávka. Hlavně
nesmí býti vpraveno málo lisovací směsi. Povrch formy má vždy
býti lesklý, hladký. Byla-li forma přehřáta, musí býti její povrch
před novým užitím přeleštěn. Připékání výlisků na povrch forem
se zabrání vymazáním formy vaselinou, parafinem, kyselinou stea-
rovou a pod. Za stejným účelem mohou býti k lisovacím směsím
přidány nepatrné přísady stearinu, kyseliny stearové, vosku,
vaseliny a jiných mastných látek.
